Gameface

GameFace is a social media platform for gamers that uses HTML, CSS, and JavaScript for the front-end, and Java with Spring Boot, Amazon RDS, and Amazon S3 for the back-end. It features Google authentication, RAWG API integration for showcasing favorite games, and Filestack API integration for social features like posts, likes, comments, and highlights. GameFace allows users to find gaming partners based on various criteria, creating a safe and inclusive gaming environment

Rocket Roasts Coffee

Rocket Roasts Coffee is an exciting paired programming web application that leverages CSS, HTML, and JS to create a dynamic coffee shop website. This innovative platform enables users to easily search through a vast selection of coffees and roasts based on either the coffee name or roast preference. Users can conveniently add or remove their preferred coffees as desired, making for a highly personalized and satisfying coffee selection experience.

Groovy Movies Project

Groovy Movies is a paired programming project that uses CSS, HTML, API's, and JS to create a movie browsing website. It features a search bar for filtering through a large movie database and displays movie ratings, descriptions, and genres. Its unique rotating card feature shows a movie poster on one side and a detailed movie description on the other, making it a fun and engaging way to discover new films.
